## How to Play
Use the arrow keys to move the tiles.  
When two tiles with the same number touch, they merge into one!

| Key | Function |
| :---: | --- |
| `↑` | Move up |
| `↓` | Move down |
| `←` | Move left |
| `→` | Move right |
